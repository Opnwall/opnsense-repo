Opnwall OPNsense Community Repository

Unofficial community plugins for OPNsense amd64 systems.

Installation

fetch -o /usr/local/etc/pkg/repos/opnwall.conf https://opnwall.github.io/opnsense-repo/opnwall.conf
pkg update -f

After updating, open System → Firmware → Plugins.
